Installing TiddlyRemember

28th September 2021 at 6:59pm
TiddlyRemember

Since TiddlyRemember integrates TiddlyWiki and Anki, it requires installation steps in both programs, so setting it up is a bit more involved than most TiddlyWiki plugins. Here's the rundown.

Step 1: Install the TiddlyWiki plugin

First you need to install the TiddlyWiki plugin into your wiki. To do so, drag and drop this link into your TiddlyWiki: TiddlyRemember.

Alternatively, if you're running on Node.js, you might prefer to (but do not have to) install the plugin as a plugin folder instead. See Installing custom plugins on Node.js in the TiddlyWiki documentation for details on the available options. You can find TiddlyRemember's code in this GitHub repository.

Step 2: Install TiddlyWiki on Node.js

You need TiddlyWiki installed on Node.js on your computer to use TiddlyRemember. You don't need to actually host your wiki on Node – you just need it to be installed to help out with the sync.

If you don't already have TiddlyWiki on Node.js installed, follow the TiddlyWiki documentation to install it. You can stop after step 4 if you're only using this to run TiddlyRemember.

If you're using a Mac and you installed using Homebrew, there may be one more step – see Calling TiddlyWiki in MacOS.

Step 3: Install the Anki plugin

  1. In Anki, choose Tools > Add-ons > Get Add-ons.
  2. Paste the add-on's code, 60456529, into the box and click OK.
  3. Quit and restart Anki.

If you prefer to see what you're downloading before you install it, you can first visit the Anki add-on's homepage.
